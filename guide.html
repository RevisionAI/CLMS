Client Management System User Guide

Introduction

Welcome to the Client Management System, a comprehensive tool designed to help you manage clients, collect their training needs, and generate customized scenario simulations using advanced AI models. This guide will walk you through the features of the application and provide step-by-step instructions on how to use them effectively.

Getting Started

Overview of the Interface

Upon launching, the application is divided into two main sections:

  • Sidebar: Used for navigating between clients, forms, and conversations, as well as selecting AI models.
  • Main Page: Displays detailed information and interactive elements based on your selections in the sidebar.

Managing Clients

Adding a New Client

  1. Access the Sidebar: Locate the sidebar on the left side of the application.
  2. Add New Client: Click the "Add New Client" button.
  3. Enter Client Details:
    • Client Name: Input the client's name in the text field.
    • Upload Image (Optional): Click "Browse files" to upload an image of the client.
  4. Save Client: Click the "Save Client" button to add the client to the system.
  5. Confirmation: A success message will confirm the client has been added.

Uploading Client Images

While adding a new client or editing an existing one, you can upload an image to personalize the client profile.

Supported image formats: .jpg, .jpeg, .png.

Managing Client Forms

Creating a New Form

  1. Select a Client: Click on a client's name in the sidebar to select them.
  2. Create New Form: In the "Client Forms" section, click "Create New Form".
  3. Form Title: Enter a title for the form or use the default timestamped title.
  4. Create Form: Click the "Create Form" button to generate the form.

Collecting Client Information

Once a form is created:

  1. Business Needs and Goals: Answer questions related to the client's business challenges and desired outcomes.
  2. Learner Profile and Needs: Provide details about the target audience for the training.
  3. Training Preferences: Specify preferences for training methods and participant engagement.
  4. Submit Information:
    • Fill in the text areas provided for each question.
    • Click "Submit" at the bottom of the form to save the responses.
  5. Confirmation: A message will confirm that the information has been saved.

Viewing Collected Information

Navigate to the "Collected Information" section on the main page.

View the table displaying all questions and the corresponding answers provided. This information is used by the AI models to generate tailored recommendations and simulations.

Conversations

Starting a New Conversation

  1. Select a Form: Ensure you have a form selected for the client.
  2. Conversations Section: Scroll to the "Conversations" area.
  3. Start New Conversation: Click "Start New Conversation".
  4. Conversation Title: Enter a title or use the default timestamped title.
  5. Create Conversation: Click the "Create Conversation" button.
  6. Conversation Interface: The chat interface will appear on the main page.

Chatbot Interaction

  • Chat Input: Use the text input at the bottom of the chat interface to type your messages.
  • Send Message: Press Enter or click the "Send" button.
  • AI Response: The AI assistant will process your query and provide a response.
  • Conversation History: All messages are saved and displayed chronologically.

Selecting AI Models

Model Selection: In the sidebar, under "Select AI Model", choose from the available AI models:

  • Ollama
  • Groq
  • GPT-4o-mini
  • Claude 3.5 Sonnet

Different models may provide varying styles of responses. Experiment to find the best fit for your needs.

Scenario Simulation Builder

Generating Scenario Simulations

  1. Access the Scenario Simulation Builder: Click on the "Scenario Simulation Builder" tab at the top of the main page.
  2. Enter a Topic: Provide a specific topic related to the client's training needs in the input field.
  3. Generate Scenario: Click the "Generate Scenario Simulation" button.
  4. Processing: The system will fetch the latest news related to the topic and generate a simulation scenario.
  5. View Scenario: The generated scenario will be displayed under "Generated Scenario Simulation".

Creating Group Activities

  • Generate Group Activity: Click the "Create Group Activity" button.
  • Output: A detailed group activity plan will be generated based on the topic and training preferences.
  • View Activity: The group activity will be displayed under "Generated Group Activity".

Generating Solutions

  • Get Solution: After generating a scenario, click the "Get Solution" button.
  • Processing: The AI model will provide the best possible solution to the challenges presented in the scenario.
  • View Solution: The solution will be displayed under "Solution".

Frequently Asked Questions

Q1: Do I need API keys to use the AI models?

Answer: Yes, some AI models require API keys (e.g., OpenAI, Anthropic). Ensure you have valid keys and they are set in your environment variables.

Q2: Can I edit client information after submitting?

Answer: Yes, you can revisit the form, update the responses, and resubmit to save changes.

Q3: How is the client data stored?

Answer: All client data is stored securely in a local SQLite database within the application directory.

Troubleshooting

  • Database Connection Error: If you encounter a database connection error, ensure that the application has permission to create and write to files in its directory.
  • API Errors: If the AI models are not responding:
    • Check your internet connection.
    • Verify that your API keys are correctly set in the environment variables.
    • Ensure that the APIs are not experiencing downtime.
  • Image Upload Issues: If you're unable to upload images:
    • Confirm that the image file is in a supported format.
    • Check that the client_images folder exists and is writable.
